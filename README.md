## YouTube访问方式

### @Handle格式（推荐）
使用@handle可以直接访问频道：
- 格式：`https://www.youtube.com/@[handle]`
- 例如：`https://www.youtube.com/@laoshi_tec`

### 频道ID格式
使用频道ID访问：
- 格式：`https://www.youtube.com/channel/[CHANNEL_ID]`
- 例如：`https://www.youtube.com/channel/UC5mVFJf71Ax6TJZcbmTnilw`
